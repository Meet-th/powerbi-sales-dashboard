# Power BI Sales Dashboard

## Overview
This project is a Power BI dashboard designed to analyze sales performance across regions, products, and time periods.

## Key Features
- Sales trends over time
- Regional performance analysis
- Top-performing products
- KPIs: Revenue, Growth %, Order Count

## Tools Used
- Power BI Desktop
- DAX
- Excel / CSV (data source)

## Data Source
Sample sales dataset (anonymized).

## How to Use
1. Download the `.pbix` file
2. Open it using Power BI Desktop
